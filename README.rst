Hack to enter, bury, unbury gnus.

Enhanced and packaged this `hack <http://www.emacswiki.org/emacs/SwitchToGnus>`_ from emacswiki.

Key bindings
============
There is only one key binding available, of course you can customize that (see
``door-gnus-key``).

::

   f9  g  door-gnus
